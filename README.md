# Beego_CRUD
CRUD for Beego Framework + Faker Library

# Depedencies
- Golang 
- Postgres
- https://github.com/icrowley/fake

# Configuration

Change the enviroment data in conf/app.conf

- ConnectionType = postgres
- Username = username
- Password = pass
- Host    = 127.0.0.1
- Port    = 5432
- DBName  = beego
- SSLMode = disable
- TotalFaker = 30 ( iteration of faker data in your application)
